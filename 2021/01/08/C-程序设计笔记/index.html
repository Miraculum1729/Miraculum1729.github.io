面向对象？

C++是所谓的Object-Oriented Programming，orient查词典，其实并没有“面向”的意思，它是“源于”。在台湾翻译作“物件导向”，要从对象来出发来分考虑问题、解决问题。

什么是对象？

  • Object = Entity (一切可以拿来讨论的东西：一本书、一句诗、一抹微笑)
  • Object may be
    • Visible or
    • invisible
  • Object is variable in programming language. 变量其实就是对象

Objects = Attributes + Services

  • Data：the properties or status （不同的对象有不同的性质、地位）
  • Operations：the functions （不同的对象可以提供的服务也不一样）

编程的目的是什么？

把我们要解决的问题通过计算机的程序描述出来。

Mapping：

  • From the problem space to the solution one.
